# Mobile UX タイポグラフィ運用ルール

## 目的
- 見出し/本文/補足の階層を統一し、画面ごとの情報密度を安定させる。

## タイプ定義（ThemedText）
- **display**: 最上位のヒーロー見出し（1画面1つまで）
- **title**: 画面タイトル／大見出し
- **headline**: セクション見出し
- **subtitle**: サブセクション／補助見出し
- **body**: 本文（デフォルト）
- **bodySemiBold**: 強調本文（ラベルや数値）
- **caption**: 注釈・補足・メタ情報
- **link**: CTAリンク（本文から独立させて使用）

## 運用ルール
1. 1画面の階層は **最大3段**（例：title → headline → body）。
2. セクションタイトルは `headline`、補足は `caption` に統一する。
3. 数値の強調は `bodySemiBold` を使用し、色は `textPrimary` のままにする。
4. 文章量が多い場合は `body` を基本にし、`subtitle` は多用しない。
5. リンクは `link` で統一し、通常本文と混在させない。
